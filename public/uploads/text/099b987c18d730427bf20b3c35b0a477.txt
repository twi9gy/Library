Я создал репозиторий в bitbucket, сгенерировал код. Теперь надо каждому скопировать себе в новую папку проект:
 
Для этого:
 
1) Создайте папку под репозиторий
 
2) скопируйте себе репозиторий
git clone https://twi9gy@bitbucket.org/twi9gy/clinic.git ./
 
3) создаем новую ветку
 
$ git checkout -b Название ветки
 
4) Публикуем ее на bitbucket
 
$ git push origin Название ветки
 
5) каждому члену команды надо взять одну из функций Api и изменить ее. Для этого сначала надо создать в ISSue Tracker задание типа некорректно работает функция и т.п.
 
6) Изменяем функцию и делаем комит. Для этого добавляем файлы 
$ git add <file1> <file2> ...
 
делаем комит:
$ git commit -m 'Комментарий к коммиту'
 
Завиваем код на ветку
$ git pull origin Название ветки  
$ git push origin Название ветки
